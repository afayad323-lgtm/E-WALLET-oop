
# 💳 Mini E-Wallet System (Java OOP)

A simple console-based E-Wallet system built using Java and Object-Oriented Programming concepts.

---

## 🚀 Features

- Create multiple users
- Deposit money
- Withdraw money with validation
- Check balance
- Search user by name
- Display all users

---

## 🧠 Concepts Used

- Classes & Objects
- Encapsulation (private fields + getters/setters)
- ArrayList
- Methods
- Loops
- Conditional statements (if / switch)
- Basic search algorithm

---

## 📁 Project Structure

User.java → User model (name, balance, operations)
Main.java → Menu system + program logic

---

## ▶️ How to Run

1. Open project in IntelliJ IDEA or any Java IDE  
2. Run `Main.java`  
3. Use the menu in console:
    1- Deposit
    2- Withdraw
    3- Show Balance
    4- Show all users
    5- Exit
